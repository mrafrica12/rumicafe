Connect this portal to your Google Apps Script backend.

API Configuration
Deploy rumi-cafe-backend.gs as a Web App in Google Apps Script, then paste the URL and token below.
Setup Guide
  1. Open Google Sheets → create a new spreadsheet named Rumi Café Operations.
  2. Go to Extensions → Apps Script.
  3. Paste rumi-cafe-backend.gs into Code.gs and save.
  4. Run setupRumiCafeBackend() and approve permissions.
  5. Go to Deploy → New deployment → Web App.
    Execute as: Me · Access: Anyone. Copy the URL.
  6. Open the Settings sheet → copy the Portal Token value.
  7. Paste both into the form on the left and click Save Settings.
  8. Click Test Connection — you should see a success message.
all set — enjoy running the café. ☕
